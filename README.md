# 🍅 桌面番茄钟应用

一个优雅的 Electron 桌面番茄钟应用，帮助你提高工作效率。

## 功能特点

- ✅ **基础计时器**：默认工作 25 分钟 + 休息 5 分钟
- ✅ **自定义时长**：可设置工作和休息时长
- ✅ **任务管理**：记录当前任务内容
- ✅ **统计功能**：今日完成的番茄数量和工作时长
- ✅ **历史记录**：查看最近 7 天的番茄记录
- ✅ **系统通知**：工作和休息结束时提醒
- ✅ **声音提醒**：计时结束时播放提示音

## 安装和运行

### 1. 安装依赖
```bash
npm install
```

### 2. 运行应用
```bash
npm start
```

## 使用说明

### 开始计时
1. 在"当前任务"输入框中输入你正在做的事情
2. 点击"开始"按钮开始计时
3. 计时过程中可以点击"暂停"暂停计时
4. 点击"重置"可以重新开始

### 自定义设置
1. 点击"⚙️ 设置"展开设置面板
2. 设置工作时长（1-60 分钟）
3. 设置休息时长（1-30 分钟）
4. 选择是否启用声音提醒和系统通知

### 查看统计
- 应用主界面显示今日完成的番茄数量和累计工作时间
- 点击"查看历史记录"可以看到最近 7 天的记录

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **HTML/CSS/JavaScript** - 前端技术
- **localStorage** - 本地数据存储

## 项目结构

```
pomodoro-timer/
├── package.json        # 项目配置
├── main.js             # Electron 主进程
├── preload.js          # 预加载脚本
├── src/
│   ├── index.html      # 主界面
│   ├── styles.css      # 样式文件
│   ├── renderer.js     # 渲染进程逻辑
│   └── timer.js        # 计时器核心逻辑
└── README.md           # 说明文档
```

## 数据存储

应用使用浏览器 localStorage 存储以下数据：
- 用户设置（时长、通知选项等）
- 今日统计数据
- 最近 30 天的历史记录

所有数据都保存在本地，不会上传到服务器。

## 开发说明

如需修改界面或功能，主要编辑以下文件：
- `src/index.html` - 修改界面布局
- `src/styles.css` - 修改样式和外观
- `src/renderer.js` - 修改交互逻辑
- `src/timer.js` - 修改计时器核心逻辑

## 打包发布

如需打包为独立应用，可以使用 electron-builder：

```bash
npm install --save-dev electron-builder
```

然后在 package.json 中添加打包配置。
